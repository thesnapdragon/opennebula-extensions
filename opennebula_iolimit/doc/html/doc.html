GLOBAL DEFINITIONS

COMMON: OpenNebula's helper scripts, used for log functions

Example: 

 COMMON='/var/tmp/one/scripts_common.sh'

DEVICEDIR: directory where the devices were mounted

Example: 

 DEVICEDIR='/dev/shared_lvm'

CGROUP: path to cgroup directory

Example: 

 CGROUP='/cgroup'
